\begin{abstract}
We present a fully executable neuro-symbolic prototype that maps uncertain concept memberships into Narsese truth values and delegates inference to OpenNARS for Applications (ONA). We focus on a controlled fit/disambiguation domain and evaluate six explicit ablations under held-out noun-pair generalization. The key empirical result is that ONA-based belief revision and multi-hop chaining are both necessary: direct propagation baselines saturate around $0.59$ accuracy, revision-only reaches $\sim0.92$, and multi-hop+revision reaches $1.00$ on the current benchmark. All reported numbers are generated from runnable code in this repository with confidence intervals and paired significance tests.
\end{abstract}

\section{Scope and Claim}
This work is a controlled benchmark study, not a claim of broad Winograd-scale natural language understanding. The contribution is methodological and empirical:
\begin{enumerate}
    \item deterministic bridge from soft concept signals to Narsese truth values,
    \item executable ONA reasoning loop (including contradiction handling),
    \item ablation protocol separating direct propagation, revision, and multi-hop reasoning.
\end{enumerate}

\section{Method}
For each sentence, the bridge emits statements of the form:
\[
\langle adjective \rightarrow concept \rangle.\ \%f;c\%
\]
where $f$ and $c$ are bounded frequency/confidence values. ONA then reasons over background rules and optional conflict context.

We evaluate three ONA variants:
\begin{itemize}
    \item \textbf{Direct propagation}: $\langle large\_like \rightarrow subject\_cause \rangle$ and $\langle small\_like \rightarrow object\_cause \rangle$ only.
    \item \textbf{Revision}: direct propagation + contradictory context (e.g., $\langle large \rightarrow small\_like \rangle$).
    \item \textbf{Multi-hop + revision}: revision plus additional latent chains (e.g., $\langle compact\_like \rightarrow small\_like \rangle$).
\end{itemize}

\section{Evaluation Protocol}
We generate a deterministic synthetic benchmark with 60 noun pairs ($360$ total examples). Each pair contributes:
\begin{itemize}
    \item lexical core examples,
    \item synonym generalization examples,
    \item conflicting evidence examples,
    \item multi-hop chain examples.
\end{itemize}

Train/test split is done by noun pair (50/50), so test examples use unseen noun identifiers. Neural baselines are trained only on training rows and evaluated on held-out rows. We report:
\begin{itemize}
    \item test accuracy,
    \item 95\% bootstrap confidence intervals,
    \item paired McNemar exact tests against the strongest ONA condition.
\end{itemize}

Scenario-wise behavior:
\begin{itemize}
    \item conflicting\_evidence: only revision-enabled ONA variants solve this (\(1.00\)).

McNemar exact tests vs ONA multi-hop+revision are significant for all comparisons in this run (largest \(p=4.88\times10^{-4}\)).

\subsection{Split Robustness}
Across five noun-heldout split seeds (0..4):
\begin{itemize}
    \item exact lexical baseline: mean \(0.487\), std \(0.008\)
    \item embedding-only bridge: mean \(0.589\), std \(0.010\)
    \item neural baseline (MLP ensemble): mean \(0.584\), std \(0.011\)
    \item ONA direct propagation: mean \(0.589\), std \(0.010\)
    \item ONA revision: mean \(0.922\), std \(0.010\)
    \item ONA multi-hop+revision: mean \(1.000\), std \(0.000\)
\end{itemize}

\section{External Benchmark Check}
We added an offline evaluation on the cached Winograd WSC273 set. On the full 273 examples, current neural baselines remain stronger than the present symbolic bridge:
\begin{itemize}
    \item SentenceTransformer replacement: \(0.498\)
    \item GPT-2 sentence score: \(0.524\)
    \item GPT-2 medium sentence score: \(0.549\)
\end{itemize}
For a strict executable causal subset (\(24\) examples), replacing handcrafted mappings with a learned LOPO bridge improves from \(0.542\) (descriptor-centroid / static ONA mapping) to \(0.625\) (learned bridge + ONA). This is still small-sample and not yet sufficient for broad external claims, but it demonstrates a concrete path away from handcrafted bridge behavior.

\section{Limitations}
\begin{itemize}
    \item External benchmark integration exists, but current transfer performance is not yet competitive.
    \item Semantic grounding is still hand-calibrated at the concept level.
    \item The current neural baseline is a compact offline MLP, not a large pretrained LM.
\end{itemize}

\section{Conclusion}
The current repository now supports reproducible, non-mocked evidence that ONA-style revision and multi-hop symbolic inference provide measurable gains over lexical, embedding-only, and compact neural baselines in this controlled setting. The next step toward top-tier claims is transfer to externally accepted datasets with similarly strict no-mock protocols.
